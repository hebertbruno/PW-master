	Closure é um encapsulamento de funções. 
	
	Uma função pode ver tudo o que tem dentro de si, inclusive o que outras funçoes ali dentro fazem, porém não podem enxergar para fora do seu próprio campo.
	
	No codigo descrito, a função foo() é declarada dentro de uma função global (para o trecho de codigo), logo essa função Global tem acesso a tudo dentro de foo(). Como "moo" foi declarado dentro da função global, ele poderá fazer uso da função foo().
	
	Supondo que essa nao fosse a função global, fosse apenas uma função f() e "moo" fosse declarada fora e quisesse utilizar foo(), daria um erro, pois a função foo() não seria reconhecida.

(function() {
	function foo(x) {
		var baz = 3;
		return function (y) {
			console.log(x + y + (++baz));
		}
	}
	var moo = foo(2); // moo agora é um closure.
	moo(1); // 7
	moo(1); // 8
})();
	moo = foo(5) // moo nao teria acesso a foo e o console diria que foo() não foi definido.

A Linguagem HTML

HTML (abreviacao para a expressao inglesa HyperText Markup Language, que significa Linguagem de Marcação de Hipertexto) é uma linguagem de marcação utilizada na contrução de páginas na Web.

Documentos HTML podem ser interpretados por navegadores.

Recurso Tags Associadas Descrição
Links <a> criar links entre as paginas
Listas <ul> e <li> Criar listas de itens ordenados
<ol> e <li> Criar Listas de itens não ordenados
Imagens <img> Inserir Imagens
